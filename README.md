# 🎮 Tic-Tac-Toe

A simple browser-based Tic-Tac-Toe game built with JavaScript modules and factory functions.

Two players enter names, take turns, and battle for eternal glory (or at least bragging rights until tea time). ☕

## ✨ Features

- 👥 Custom player names
- 🔁 Turn-based play
- 🏆 Winner detection (rows, columns, diagonals)
- 🤝 Tie detection
- 🧱 Dynamic board rendering in the DOM

## 🛠️ Tech Used

- HTML
- JavaScript
- Tailwind utility classes (in the markup)

## ▶️ How To Run

1. Open `index.html` in your browser.
2. Enter player names.
3. Click **Start Game**.
4. Click a square to place your mark.
